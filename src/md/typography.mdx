### Heading, Heading.Light and Heading.ExtraBold

Heading component. Ligth and ExtraBold heading affect only font-weight

### Props

+-----------------+---------------+-------------------+----------------------------------------------------------------+
| Prop            | Type          | Default           | Notes                                                          |
+=================+===============+===================+================================================================+
| level           | number        |   2               | Determines which "h" tag should be used.                       |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| color           | string        | #FFFFFF           | Determines heading color.                                      |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| size            | string        |   small: "tera"   | If not defined heading is responsive. When size is defined     |
|                 |               |   medium: "zetta" | heading is uses defined size in all resolutions                |
|                 |               |   large: "bronto" | Supports all typography values                                 |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| responsiveSize  | spacingObject | {}                | Add resposive text size.                                       |
|                 |               |                   | String values are the same as size prop                        |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| lineHeight      | string        | "body"            | Determines text lineHeight.                                    |
|                 |               |                   | Possible values: "body" and "header"                           |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| children        | node          | null              | Render as the Text's children                                  |
+-----------------+---------------+-------------------+----------------------------------------------------------------+
| ...props        | any           |                   | Passes through any other props to underlying div, or in        |
|                 |               |                   |  the case of a different tag prop being passed, other element  |
+-----------------+---------------+-------------------+----------------------------------------------------------------+

spacingObject = {
  small: String,
  medium: String,
  large: String
}

### Text

Text Component with a dark, light, and accent variants available.

### Props

+-----------------+---------------+---------+--------------------------------------------------------------------------+
| Prop            | Type          | Default | Notes                                                                    |
+=================+===============+=========+==========================================================================+
| tag             | string        |   div   | Determines the tag of the underlying element.                            |
|                 |               |         |  One of "div", "span" and "p".                                           |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| variant         | string        |   dark  | Determines the style of the Text.                                        |
|                 |               |         | One of "dark", "light", and "accent".                                    |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| size            | string        |   hecto | Determines the font-size of the Text. One of                             |
|                 |               |         | "mini", "uno", "hecto", "kilo", "giga", and "tera".                      |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| accent          | string        |    ""   | If variant equals "accent", this props determines the style of the Text. |
|                 |               |         | One of "aquamarine", "azure", "alert", "caution", "cruz", "heliotrope",  |
|                 |               |         |"positive", "summerSky", and "turquoise".                                 |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| weight          | string        | regular | Determines the font-weight of the Text. One of "regular" and "semiBold". |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| primary         | boolean       | false   | Determines if the base hue of the font-color should be used.             |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| secondary       | boolean       | false   | Determines if the light hue of the font-color should be used.            |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| disabled        | boolean       | false   | Determines if the muted hue of the font-color should be used.            |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| allCaps         | boolean       | false   | Determines if the text is displayed with capital letters.                |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| children        | node          | null    | Render as the Text's children                                            |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| responsiveSize  | spacingObject | {}      | Add resposive text size. String values are the same as size prop         |
+-----------------+---------------+---------+--------------------------------------------------------------------------+
| ...props        | any           |         | Passes through any other props to underlying div, or in the case of      |
|                 |               |         | a different tag prop being passed, other element                         |
+-----------------+---------------+---------+--------------------------------------------------------------------------+

spacingObject = {
  small: String,
  medium: String,
  large: String
}
