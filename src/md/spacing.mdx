### Spacing

Spacing component is a wrapper component that adds top/bottom padding and allows the developer
to define spacing behavour on different screen sizes

### Props

+-----------+---------------+---------+------------------------------+
| Prop      | Type          | Default | Notes                        |
+===========+===============+=========+==============================+
| top       | spacingObject |   {}    | Sets top padding             |
+-----------+---------------+---------+------------------------------+
| bottom    | spacingObject |   {}    | Sets bottom padding          |
+-----------+---------------+---------+------------------------------+

spacingObject = {
  xSmall: String,
  small: String,
  medium: String,
  large: String
}
